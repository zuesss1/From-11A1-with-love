 NEW YEAR FIREWORKS SHOW
Một ứng dụng web trình chiếu pháo hoa nghệ thuật với hiệu ứng Neon, đồng hồ đếm ngược hệ kỹ thuật số và khả năng tương thích hoàn hảo trên cả máy tính lẫn điện thoại.

 TÍNH NĂNG CHÍNH
Hiệu ứng quả địa cầu Pháo nổ tạo hình cầu 3D kèm vành đai ánh sáng (Saturn Style).

Đồng hồ đếm ngược Đếm ngược 5 giây trước giờ G với định dạng Ngày  Giờ  Phút  Giây.

Chữ Neon nghệ thuật Dòng chữ HAPPY NEW YEAR hiện ra mềm mại với hiệu ứng ánh sáng mờ ảo.

Tương thích đa thiết bị Tự động điều chỉnh kích thước hạt và mật độ pháo phù hợp cho PC và Mobile.

Công nghệ Sử dụng HTML5 Canvas nguyên bản, không dùng thư viện nặng nề.

 CẤU TRÚC THƯ MỤC
Dự án này cực kỳ gọn nhẹ chỉ với một file duy nhất

Plaintext

└── index.html      # Chứa toàn bộ logic HTML, CSS và JavaScript
 HƯỚNG DẪN CÀI ĐẶT
Tải file index.html về máy tính của bạn.

Mở file bằng bất kỳ trình duyệt nào (Chrome, Edge, Safari).

Để thay đổi ảnh nền lúc đếm ngược, tìm dòng có #countdown-bg trong code và thay link ảnh tại mục url(...).



 GIẤY PHÉP
Dự án này được tạo ra cho mục đích giải trí và học tập. Bạn có thể tự do chỉnh sửa và chia sẻ cho bạn bè.

Chúc bạn một năm mới rực rỡ như những ánh pháo hoa!
